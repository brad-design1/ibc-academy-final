## COLOR SYSTEM FIXES

### Current Problems:
- `bg-gold text-gold` combinations (unreadable)
- Hover states that reduce contrast instead of increasing it

### Proposed Color System:

```
PRIMARY BACKGROUNDS:
- bg-[#050A12]     ‚Äî deepest black (body)
- bg-[#0A1628]     ‚Äî dark navy (sections)
- bg-[#0F1F35]     ‚Äî medium navy (cards)
- bg-[#1B365D]     ‚Äî navy (accents)

TEXT:
- text-white        ‚Äî headings, primary text
- text-[#94A3B8]    ‚Äî body text (slate-400)
- text-[#CBD5E1]    ‚Äî secondary text (slate-300)

ACCENT:
- bg-[#C9A84C]      ‚Äî gold (buttons, badges)
- text-[#0A1628]    ‚Äî dark text ON gold buttons (high contrast)
- hover:bg-[#D4B96A] ‚Äî lighter gold on hover (KEEP dark text)

BUTTON STYLES:
- Primary: bg-gold text-navy-900 ‚Üí hover:bg-gold-light text-navy-900
  (Gold bg, dark text ‚Äî ALWAYS readable)
- Secondary: border-gold text-gold ‚Üí hover:bg-gold/10 text-gold
  (Ghost button ‚Äî gold border and text)
- Tertiary: border-slate-600 text-white ‚Üí hover:border-gold text-gold
  (Subtle ghost ‚Äî white to gold transition)
```

### Button Contrast Fix (WCAG AA):
- Gold (#C9A84C) on Navy-900 (#0A1628) = **ratio 7.2:1** ‚úÖ
- Navy-900 (#0A1628) on Gold (#C9A84C) = **ratio 7.2:1** ‚úÖ  
- White (#FFFFFF) on Gold (#C9A84C) = **ratio 2.1:1** ‚ùå (FAIL)
- **NEVER use white text on gold background**
- **ALWAYS use dark navy text on gold buttons**

## TYPOGRAPHY UPGRADE

Current: Inter (clean but generic)

Proposed:
- **Headings:** "PP Neue Montreal" or "Satoshi" (modern geometric, premium feel)
  - Fallback: Inter with tighter letter-spacing and heavier weight
- **Body:** Inter stays (excellent readability)
- **Accents/Quotes:** Georgia or serif for pull quotes (warmth + authority)

If we can't use premium fonts, tweak Inter:
- Headings: font-black (900), letter-spacing: -0.02em, leading-tight
- This alone adds significant visual weight

## INTERACTION & MOTION

Subtle, purposeful animations:
- **Scroll-triggered fade-ins** ‚Äî sections appear as you scroll (stagger children)
- **Headshot parallax** ‚Äî hero headshots move slightly on scroll
- **Card hover lift** ‚Äî cards rise 4px with soft shadow on hover
- **Tab underline slide** ‚Äî content category tabs have sliding underline
- **Counter animations** ‚Äî "50+ articles" counts up when visible
- **Smooth scroll** ‚Äî anchor links glide

**DO NOT:**
- Auto-play video
- Use loading spinners as decoration
- Add particle effects
- Animate everything at once

Keep it Masterclass-level: elegant restraint.
